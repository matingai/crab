# Support

Crab is in active 0.1.x development. The best support channel depends on the type of
request.

## Where To Ask

- **Bug reports**: open a GitHub issue with the bug report template.
- **Feature ideas**: open a feature request or start a design discussion.
- **Security concerns**: follow [SECURITY.md](SECURITY.md) and avoid public disclosure.
- **Contribution questions**: open a draft PR or discussion with the design context.
- **Documentation gaps**: open a docs issue with the page or command that confused you.

## Good Reports

Useful reports include:

- the command or workflow you ran;
- OS, Rust version, Node version, and model provider when relevant;
- logs or screenshots with secrets removed;
- expected behavior and actual behavior;
- whether local files, browser state, Office documents, or shell tools were involved.

## Boundaries

The maintainers cannot review private credentials, private workspaces, proprietary model
logs, or confidential third-party documents. Please reduce issues to a small reproduction
before sharing.
